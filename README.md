# expand_disk
Скрипт для увеличения раздела диска (MBR/GPT/LVM)
## Как запустить:

```bash
chmod +x expand_disk.sh
sudo ./expand_disk.sh
```

## Скрипт:

Проверяет наличие свободного места после последнего раздела (и сообщает, если нет).

Запрашивает диск и раздел на одной строке с пользователем.

Обрабатывает ситуацию, когда расширять некуда (диск не увеличен).

Поддерживает обычные разделы и LVM, перемещение swap.
